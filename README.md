# NOR Gate using Verilog

## Overview

This project implements a **2-input NOR Gate** in Verilog HDL along with a testbench for simulation. The design can be simulated on **EDA Playground**, and the output waveform can be viewed using **EPWave**.

## Features

* 2-input NOR gate implementation
* Verilog HDL design
* Testbench with all input combinations
* EPWave-compatible waveform generation
* Beginner-friendly and easy to understand

## Files

* `nor_gate.v` – Verilog implementation of the NOR gate
* `nor_gate_tb.v` – Testbench for simulation and waveform generation

## Simulation

1. Open EDA Playground.
2. Paste the design code into the **Design** window.
3. Paste the testbench into the **Testbench** window.
4. Select **Icarus Verilog** as the simulator.
5. Enable **Open EPWave after run**.
6. Click **Run** to simulate and view the waveform.
